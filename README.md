# Multi-Purpose Utility Web Application

## 📖 Project Overview
This project is a versatile frontend web application that houses several different utility tools and calculators within a single interface. It was built to demonstrate proficiency in frontend structure, user interface design, and implementing mathematical and business logic through code.

## ✨ Key Features
* **Home Page:** A central landing hub that provides easy navigation to the various tools.
* **Length Conversion Tool:** Accurately converts values between meters and feet (and vice versa).
* **Temperature Conversion Tool:** Converts temperatures seamlessly between Celsius and Fahrenheit (and vice versa).
* **Factorial Calculator:** A mathematical utility that computes the factorial of user-inputted numbers.
* **Mini Payroll System:** A foundational business tool designed to calculate basic employee wages, deductions, and net pay.

## 💻 How to Run Locally
Because this project does not require a backend server, running it on your own machine is incredibly simple:
1. Clone this repository or download the ZIP file.
2. Extract the folder to your computer.
3. Double-click the `index.html` file to open it directly in your preferred web browser.
